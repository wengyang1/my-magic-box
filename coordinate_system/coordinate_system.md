# 坐标系
```text
坐标系的英文是 "coordinate system"。这个词组描述了用于确定空间中点的位置的数学系统，
它通常由一组有序的实数或复数（称为坐标）组成，这些坐标根据特定的规则与空间中的点相对应。
在几何学和物理学中，坐标系是非常重要的工具，它允许我们精确地描述和计算空间中的物体和事件。
常见的坐标系包括
笛卡尔坐标系（Cartesian coordinate system）、
极坐标系（polar coordinate system）、
圆柱坐标系（cylindrical coordinate system）、
球坐标系（spherical coordinate system）等。
例如，在二维笛卡尔坐标系中，一个点的位置由两个坐标（通常称为x和y）确定，
它们分别表示该点沿两个垂直轴（通常是水平轴和垂直轴）的位移。
在三维笛卡尔坐标系中，
一个点的位置由三个坐标（x、y和z）确定，
它们分别表示该点沿三个互相垂直的轴（通常是水平轴、垂直轴和垂直于这两者的轴）的位移。
因此，"coordinate system" 是描述这种用于定位和描述空间中点的数学框架的英文术语。
```
# Homogeneous Coordinates
```text
齐次坐标系（Homogeneous Coordinates）是一个在投影几何中使用的坐标系统，它使用一个n+1维的向量来表示一个原本是n维的向量或点。这种表示方法在数学、计算机图形学、机器人学及自动驾驶等领域中具有广泛的应用。以下是对齐次坐标系的详细解释：

一、定义与表示方法
定义：齐次坐标是将一个n维的向量或点扩展为一个n+1维的向量来表示。在投影几何中，这种坐标系统能够方便地表示包括无穷远点在内的所有点。
表示方法：
在二维空间中，一个点(x, y)的齐次坐标可以表示为(x’, y’, w)，其中w通常设为1，此时(x’, y’)与(x, y)等价，即x’ = x/w, y’ = y/w。
在三维空间中，一个点(x, y, z)的齐次坐标则表示为(x’, y’, z’, w)。
二、性质与特点
非唯一性：对于同一个点或向量，其齐次坐标不是唯一的。具体来说，如果(x’, y’, z’, w)是一个点的齐次坐标，那么对于任意非零实数k，(kx’, ky’, kz’, kw)也是该点的齐次坐标。
规模不变性：齐次坐标具有规模不变性，即乘以一个非零常数后，仍表示同一个点或向量。
表示无穷远点：齐次坐标能够表示无穷远点，这是笛卡尔坐标无法做到的。在齐次坐标下，当w=0时，表示的点即为无穷远点。
三、应用
计算机图形学：齐次坐标是计算机图形学中的重要工具，它能够统一平移、旋转和缩放等几何变换的矩阵表示，使得这些变换的计算更加简单和高效。同时，齐次坐标还具有表示无穷远点等独特优点，使得它在处理复杂几何问题时具有更大的灵活性和通用性。
机器人学：在机器人学中，齐次坐标也用于描述机器人的位姿和运动轨迹，以及进行空间变换和路径规划等任务。
自动驾驶：在自动驾驶领域，齐次坐标被用于地图匹配、路径规划和车辆定位等任务中，以提高自动驾驶系统的精度和可靠性。
四、与其他坐标系统的关系
与笛卡尔坐标的关系：齐次坐标是笛卡尔坐标的扩展和补充。在齐次坐标下，当w=1时，可以方便地转换回笛卡尔坐标。同时，齐次坐标能够表示笛卡尔坐标无法表示的无穷远点。
与射影坐标的关系：射影坐标是齐次坐标在射影几何中的特例。在射影几何中，点的齐次坐标表示为一个n+1维的向量，其中n是射影空间的维数。射影坐标具有齐次性，即乘以一个非零常数后仍表示同一个点。
综上所述，齐次坐标系是一个在数学和多个工程领域中具有广泛应用和重要意义的坐标系统。它不仅能够表示包括无穷远点在内的所有点，还能够统一平移、旋转和缩放等几何变换的矩阵表示，使得这些变换的计算更加简单和高效。
```
